

### Golang Ping Graph 🚀


### Usage

```bash
go build -o ping_graph main.go
cp ping_graph /usr/local/bin/

####
❯ ./ping_graph            
Usage: ping <hostname>


❯ ./ping_graph www.baidu.com  
      
```
